# Análise da Final da Copa do Mundo 2022

Projeto de análise de dados da final da Copa do Mundo FIFA 2022 entre Argentina e França utilizando Python e Jupyter Notebook.

## Visão Geral

Este projeto explora estatísticas e métricas de desempenho da final da Copa do Mundo de 2022.
A análise foca na comparação entre as duas seleções por meio de métricas ofensivas, eficiência nos passes e precisão nas finalizações.

O projeto foi desenvolvido em Python utilizando Jupyter Notebook para análise e visualização dos dados.

## Partida

* Argentina vs França
* Final da Copa do Mundo FIFA 2022
* Estádio Lusail, Catar

## Métricas Analisadas

A análise inclui:

* Gols
* xG (Expected Goals)
* Finalizações
* Finalizações no alvo
* Passes
* Precisão de passes (%)

## Tecnologias Utilizadas

* Python
* Jupyter Notebook
* Pandas
* Statsbompy
* Matplotlib
* mplsoccer

## Objetivos do Projeto

* Comparar o desempenho das seleções na final
* Explorar técnicas de análise de dados no futebol
* Praticar visualização e análise estatística com Python
* Desenvolver um projeto de portfólio em Sports Analytics

## Exemplos de Análises

Alguns pontos explorados no projeto:

* Eficiência ofensiva das equipes
* Comparação da precisão nas finalizações
* Dominância e circulação de bola através dos passes
* Relação entre xG e gols marcados
